### powershell命令缩写
```powershell
# 位置直接在powershell中输入$profile,可以输出位置
function di {docker images}
function dp {docker ps $args}
function dc($name) {docker compose $name $args}
function ca($name) {conda activate $name}
function pms {python manage.py runserver $args}
function pm($name) {python manage.py $name}

function curlj($url) {curl -H 'Accept: application/json; indent=4' $url}
```
